HTML Character Entity References

Usage of HTML Entities

  • Help avoid rendering issues
  • Safeguard against more limited character endcoding
  • Provide characters not available on keyboard (E.g. ©)

3 Characters To Avoid

  • Instead of "<", use "& lt;" (no space between)
  • Instead of ">", use "& gt;" (no space between)
  • Instead of "&", use "& amp;" (no space between)

Symbols:

  • ©: "& copy;"
  • ": "& quot;"

Non-Breaking Space

This makes it so that words connected by it will either not wrap or wrap all together at once

  • Use: "& nbsp;"

Note: this can create extra whitespace, but you should use span and add margins instead if you want to do that
