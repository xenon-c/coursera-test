Creating Links

Attributes:

  • href: hyperlink reference
    • you can paste links to link to another webpage
    • you can use ./ to specify a file in the same folder as the current file (../ for the folder above)
    • you can use #sectionname to specify a section/div in the same file that has sectionname as it's id or name attribute
  • title: used by screenreaders that help visually impaired people get through webpage
  • target:
    • _blank: opens the page specified in a new tab

Notes

  • the <a> tag is both a block level element and an inline element at the same time
    • can be used to link an entire section to be clickable (E.g. company logo/name at top right corner - people expect to be able to click that)
