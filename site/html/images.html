Displaying Images

Attributes

  • src: specify the image you want to display
    • you can paste links to link to an external image
    • you can use ./ to specify an image in the same folder as the current file (../ for the folder above)
  • width/height: specify the dimesions (in pixels) for the image
    • tells the browser there's something there even if it takes time to load (reserves space)
    • if image link doesn't work there's still space reserved so your layout doesn't break
