Fundamental (Setup) Notes

Accessing Git Repository

  1. Click "Code" on repository and copy the link
  2. Go to terminal
  3. Type: git clone "link" (don't add the quotation marks)

Checking Git Status

  1. Go to target folder
  2. Type: git status

Switching Git branch

  1. Go to target folder
  2. Type: git checkout branchname

Activating BrowserSync

  1. Go to target folder
  2. Type: browser-sync start --server --directory --files "*"
  3. Type: browser-sync start --server --directory --files "**/*" (which tells it to watch all files within that folder as well)

Commiting To Git

  1. Go to target folder
  2. Press enter after typing each of the following in order:
    • git add . (this marks the files to be committed)
    • git commit -m "text" (this commits the files to local repository)
    • git push (this updates changes to remote repository)
