Fundamental (Setup) Notes

Accessing Git Repository

Click "Code" on repository and copy the link
Go to terminal and type:
git clone "link" (don't add the quotation marks)

Checking Git Status

Go to target folder and type
git status

Switching Git branch

Go to target folder and type
git checkout branchname

Activating BrowserSync

Go to target folder and type:
browser-sync start --server --directory --files "*"

Commiting To Git

Go to target folder and type:
git add . (this marks the files to be committed)
git commit -m "description" (this commits the files to local repository)
git push (this updates changes to remote repository)
