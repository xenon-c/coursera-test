Creating Objects & Properties

  • You can create an object using var objectName = new Object();
  • If you mention a non-existent property of an object, JS automatically creates it for you
    • E.g. objectName has no "name" property, but you can create it by doing objectName.name = "something"; and it will create the name property and assign "something" as its value
    • If you want one of the properties to be an object in it of itself and set a property within, you have to define the property as an object first, and not just straight off specifying the property within the property
  • Object literal: A method of creating objects using curly braces (i.e. var objectName = {};)
    • You can directly create properties by doing:
      var compnay = {
         name: "Facebook",
         ceo: {
            firstName: "Mark",
            favColor: "blue"
         },
      };

Retrieving Values

  • You can retreive a value from an object using dot notation: console.log(objectName.propertyName)
    • Dot notation will only work with a valid javascript variable name (you can't put spaces in variable names but you can with bracket notation)
  • You can also retrieve a value from an object using bracket notation: console.log(objectName["propertyName"]);
  • There's a special for loop that retrieves evvery property in an object:
    • for (var prop in objectName) { console.log(prop + ": " + objectName[prop]); };
