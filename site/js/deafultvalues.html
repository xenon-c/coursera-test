Handling Default Values

Notes

  • Usage of Default Value: If the value of a variable is not provided, the value of the variable will become the default value
    • You can create a default value by doing variable = variable || "default value"; (works as undefined variable coerced to false, so the default is used)(if variable given value, it is read as true and JS will just simply use that value straight off)
