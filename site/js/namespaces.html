Namespaces

  • If you have different js scripts and they declare the same variable and have a function using it, you can create a namespace to sepearte them so the first doesn't get overwritten
  • You can do this by declaring the variable and the function as a property of an object, then invoking the function by specifying the object property and using the respective property

Immediately Invoked Functions

  • You can invoke a function immediately after declaring it by wrapping the function declaration in brackets (thus making it an object), then putting brackets behind it to invoke it
  • E.g. (function (greeting) { console.log(greeting); };)(hi); - hi is the argument passed into the function (greeting variable), so this function will log hi
