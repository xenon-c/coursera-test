Arrays

Notes

  • create array by doing var array = new Array();
  • array values can vary (e.g. 1 can be string, 2 can be number, 3 can be function, etc.)
  • to invoke a function stored as an array value, put brackets after the square brackets and place the argument you need to pass (optional) inside the brackets
  • each array has a property called "length" that gives you the length of the array (how many data values there are)

Adding Values

  • to add values to the array, you can do arrayName[position] = "Value"; - where position is a number (0 = 1st data, 1 = 2nd data...), and arrayName is the name of the array variable you created
  • alternatively, you can also do var arrayName = ["value1", "value2", "value3"]; - where value1 is the 1st value, at position 0

Retreiving Values

  • you can retrieve a specific value from the array using console.log(arrayName[position]);
  • There's a special for loop that retrieves every value in an array:
    • for (var value in arrayName) { console.log(value + ": " + arrayName[value]); }; - where value can be anything (it's a temporary variable), and arrayName[name]; is retrieving the value
