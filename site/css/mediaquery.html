Media Queries

Notes

  • Media queries allow you to group styles together and target to different devices based on some criteria (e.g. by width, height, portrait, landscape etc.)
    • E.g. Phone size and computer size is different so it makes sense to design different styles for each
  • Start with base style - styles that apply regardless of the device (E.g. color of text is always blue) and then add media queries after
  • Make sure you don't overlap range boundaries if you're targeting by width
  • You can use chrome developer tools (in the browser) to test out media queries (there's even a cell phone icon and you can select orientation and device and device size)
  • Break point: the points between the range of 1 media query and another

Syntax

  • @media (criteria) {}
    • a media query always starts with @media
    • an example of critera would be (max-width: 767px) and resolves to true or false - If true, style within curly braces apply (like stylesheet within stylesheet)
    • you can combine multiple criteria with logical operators:
      • (criteria) and (criteria) - applies if both criteria are true
      • (criteria), (criteria) - applies if one criteria is true (OR operator)
