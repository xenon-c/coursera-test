The Background Property

Notes

  • background-color: - you can choose any color by name
  • background-image: url("") - you can paste a url or specify a file (wins in priority vs background-color)
  • background-repeat: - you can specify if the background image is to be repeated (vertically, horizontally, no-repeat etc.)
  • backgorund-position: - you can specify where the background image is (E.g. bottom right) (if you only specify vertical/horizontal, the other will be defaulted to center
  • background: - you can use this and place all values above into 1 declaration (overwrites every other background property)
