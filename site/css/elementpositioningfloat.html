Positioning Elements By Floating

  • floating is useful for making different columns for different sections
  • floating elements can produce very flexible layouts
  • floats are taken out of their normal document flow
  • floats don't have vertical margin collapse
  • to resume normal document flow, use the clear property

Example 1:

  • When you float elements, browser takes them out of regular document flow, (div is convinced spot of the red box originally was is empty - the red box is not considered as part of the div)
  • Regular elements have margin collapsed, however (vertical) margin of floated elements will not be collapsed (so red box is a bit lower than orange as it has top margin)
